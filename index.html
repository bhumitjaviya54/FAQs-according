FAQs

What is JavaScript ?

JavaScript, often abbrevited as JS, is a programming Language that conforms to the ECMAScript specification. JavaScript is high-level, often just-in-time compiled and multi-paradigm.

Is it easy to learn HTML and CSS ?

HTML and CSS are partners that work together to create web page. And while they might be team mates, HTML and CSS visually look different, and they function differently too. So while you'll have to learn both, it's easy to learn HTML and CSS together at the same time. They aren't hard.

What is Web Development ?

Web Development is the building and maintenance of website, it's the work that happens behind the scenes to make website look great, work fast and perform well with a seamless user experience.
